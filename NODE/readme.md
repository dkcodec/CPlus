Задача №199. НОД
===
**Условие:**

  * Даны два числа. Найти их наибольший общий делитель.
---
**Входные данные:**

  * Вводятся два натуральных числа, не превышающих 10^9.
---
**Выходные данные:**

  * Выведите НОД введенных чисел.
---
**Примеры**

  * Входные данные:
    * `9 12`
  * Выходные данные:
    * `3`
 
 ---
 *Использован алгоритм Евклида.*
